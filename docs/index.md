# PDFPO

PDFPO is a minimal, open-source macOS PDF editor built with Swift and PDFKit.

## What it does
- Open and view PDFs
- Select text and replace it (via annotations)
- Export a new PDF copy

## Get the app
Clone the repo and build from source:
```bash
git clone https://github.com/your-org/pdfpo.git
cd pdfpo
./scripts/build_dmg.sh
```
