# Contributing

Thanks for your interest in contributing to PDFPO.

## Getting started
1. Fork the repo and create your branch from `main`.
2. Run `swift run` to launch the app.
3. Make your changes and ensure the app still builds.

## Code style
- Prefer clear, small functions.
- Keep UI logic in `ContentView` and PDF logic in `PDFDocumentController`.

## Pull requests
- Describe the problem and solution clearly.
- Include screenshots or a short video for UI changes when possible.

## Reporting issues
Please include:
- macOS version
- Steps to reproduce
- A sample PDF (if possible)
